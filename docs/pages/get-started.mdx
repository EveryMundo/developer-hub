# Get Started

To start, you have different options depending on your way of learning. The following sections provide basic knowledge that will help you get the big picture.

- [Overview](/get-started/overview): What's **EveryMundo Developer Hub** and key concepts.
- [Create a Module](/get-started/create-a-module): Start creating your first module through tutorials or examples code
- [FAQ](/get-started/faq): Most frequently asked questions about the ecosystem.

You also have the [Learning](/learning) section to do an in-depth study of the platform or access the [examples repository](https://github.com/EveryMundo/registry/tree/develop/examples) and experiment using the available tools.
